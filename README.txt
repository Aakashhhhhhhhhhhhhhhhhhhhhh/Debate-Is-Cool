  STEP 1 — Install Python (one time only, takes 2 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Go to:  https://www.python.org/downloads/
  2. Click the big yellow "Download Python" button
  3. Open the downloaded file to install
  4. ⚠️  IMPORTANT: During install, TICK the box that says
         "Add Python to PATH"  — this step is critical
  5. Click Install, wait for it to finish


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  STEP 2 — Open the App
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Windows:  Double-click  👉  START_WINDOWS.bat
  Mac:      Double-click  👉  START_MAC.command

  The first time you open it, it will take 1-2 minutes
  to install required tools automatically.
  After that, it opens instantly.

  A black window will appear — this is normal, keep it open.
  The app will open in your web browser automatically.

  STEP 3 — How to Use
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. First go to  👥 Student Roster  — upload your master
     student list CSV (school, cohort, student_name)

  2. Then go to  📤 Upload CSV Data  — upload any of:
     • Participation data  → charts appear automatically
     • Teacher score data  → charts appear automatically
     • Student survey data → charts appear automatically

  3. Visit each Framework page for full analysis + PDF reports

  4. To stop the app: close the black terminal window
